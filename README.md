# 隐私链浏览器

### 环境及版本说明

* node  v16.15.0
* vue  2.6.14
* axios  0.27.2
* element-ui  2.15.9
* echarts 5.3.2
* moment 2.29.4



## Project setup
```
yarn install
or
npm install
```

### Compiles and hot-reloads for development
```
yarn serve
or
npm run serve
```

### Compiles and minifies for production
```
yarn build
or
npm run build
```

### Lints and fixes files
```
yarn lint
```
