# 隐私链浏览器

### 环境及版本说明

* node  v16.15.0
* vue  2.6.14
* axios  0.27.2
* element-ui  2.15.9
* echarts 5.3.2
* moment 2.29.4



### 安装依赖
```
yarn install
or
npm install
```

### 项目运行
```
yarn serve
or
npm run serve
```

### 打包
```
yarn build
or
npm run build
```
